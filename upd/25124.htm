Date: 24th January, 2024

Been working on the USSR for a while now -- circa 3 or 4 hours. Feeling good after finishing a really buggy release. Oh well.

I think of releasing a live/stable build today, if I can, and will test for Windows later. Not really a stable build...

Today was okay. And I also added a cool, automatic launcher for the USSR, to help me run it faster, and make my life a little easier -- a QoL feature. This also adds to the user-friendly scope of the USSR. Originally, this project was not user-friendly at all. The paths weren't covered, and the code was a mess. Basically un-readable.

I also added an automatic NGROK authentication-token verifier and adder. Also added a prerequisuite section for Windows machines: it will automatically install C-URL, Tar, and also Chocolatey in the process. This helps downloading Ngrok a lot easier, as it is as simple as navigating a CLI menu to opening it.

I have to fully clean the project, reinstall, and reconfigure it, and then it should... hypothetically work. Well. I hope it does. Ngrok is weird.
