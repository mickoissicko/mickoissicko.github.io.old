Syntax

Mix's syntax is 90% Python at the moment, but this language will be feature-packed.

Here is the syntax cheat-sheet:

out = print
 
<> = () 
;; = ""
 
if = while
 
>> = in
 
/ = range
 
@ = :
 
func = def
 
_: = #
 
#f = float
 
#i = int
 
#ch = str
 
char = __name__
 
:: = =

The syntax looks overly complicated (for no reason) but that is because the language is in development.

All of those will have a purpose.

Mux

Mux is the compilation tool for Mix.

The usage is as follows: mux

Mux is cross-platform, and is OS-specific. It generates a binary depending on whatever system you're on.

If you run Mux with Windows, it compile your MIX file into a (.exe). If you are on Mac or Linux, it will compile the binaries for those accordingly.

Hello, world!

Let's create our first ever 'Hello, world!' program in Mix.

Start by create a file ending in the file-extension '.mix'. Funnily, this is not required because I think the compiler accepts any file as long as its syntax is correct. That is to be fixed. Besides the point: create a file ending with '.mix'.

Once created, add in the following line:

out<;hello, world!;>

Save the file, and open a terminal. For Linux/Mac:

./mux myapp.mix

For Windows:

mux.exe myapp.mix

It takes a while, but it should be compiled. To run your application, open the 'dist' folder it created, and run:

./myapp
 
"myapp.exe"

Datatypes

Start by create a file ending in the file-extension '.mix'. Funnily, this is not required because I think the compiler accepts any file as long as its syntax is correct. That is to be fixed. Besides the point: create a file ending with '.mix'.

Once created, add in the following lines:


_:define our module
                
func var_and_datatype@
    out.char};)
            
_:take variable
var :: 1
                
_:take variable, but as float
var2 :: #f<2>
                
_:now as a string
var3 :: #ch<;words & numbers;>
                
var_and_datatype(var, "var")
var_and_datatype(var2, "var2")
var_and_datatype(var3, "var3")
                
func print_simple_hello_world<>@
    out<;hello, world!;>
                
print_simple_hello_world<>

Try it out!

Simple mathematics

Let's define two variables, add them, and print the sum.

_: this is a comment, under the comment is a variable set to string
variable :: #ch<"hello world">

func helloWorld<>@
    out

_: basic maths
var :: #i(1)

func addition<>@
    result :: var + var
    out
