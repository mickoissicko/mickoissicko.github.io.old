Panik!

What about it?

Panic is a cool tool for closing a process upon keystroke-detection.

Usage

Panic is easy to use. It creates a file in the working directory -- 'options.cfg' -- which stores all of your preferences. It stores keybind as well as process name in a pretty obscure format, however, it is easy to mess around until you understand.

A typical options.cfg file would look like this:

ExampleProcessName.exe
X
# Generated by Panik

The line in the end, # Generated by Panik is automatically added by the program.

Panic is written in C and C++, so the resulting executable is standalone, meaning you don't have to download any other programs to run Panik.
